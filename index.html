Secret Codes

A great way to learn is to play. (That's why kids love playing) So lets play a game. Let's make secret codes





Secret Codes

You encode a message by changing the letters around so that people cannot read the message any more. You change the letter A to a Y, the letter B to a Z, the letter C to an A, and so on. So for example, the message
 To be or not to be, that is the question!
gets changed to
 Rm zc mp lmr rm zc, rfyt gq rfc oscqrgml!
Can you figure out how I did it?
