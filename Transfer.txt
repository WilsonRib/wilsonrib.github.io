Yes, you can install Nextcloud on Debian-based systems like Debian itself or Ubuntu using the `apt` package manager. Here are simplified steps:

1. **Update Package Lists:**
   ```bash
   sudo apt update
   ```

2. **Install Required Packages:**
   ```bash
   sudo apt install apache2 mariadb-server libapache2-mod-php php-mysql php-redis php-gd php-xml php-mbstring php-zip php-intl
   ```

3. **Create a MySQL/MariaDB Database and User:**
   ```bash
   sudo mysql_secure_installation
   sudo mysql -u root -p

   # Inside MySQL/MariaDB prompt
   CREATE DATABASE nextcloud;
   CREATE USER 'nextclouduser'@'localhost' IDENTIFIED BY 'your_password';
   GRANT ALL PRIVILEGES ON nextcloud.* TO 'nextclouduser'@'localhost';
   FLUSH PRIVILEGES;
   EXIT;
   ```

4. **Download and Extract Nextcloud:**
   ```bash
   sudo wget https://download.nextcloud.com/server/releases/latest.tar.bz2
   sudo tar -xvjf latest.tar.bz2 -C /var/www/
   ```

5. **Set Permissions:**
   ```bash
   sudo chown -R www-data:www-data /var/www/nextcloud/
   ```

6. **Configure Apache:**
   ```bash
   sudo nano /etc/apache2/sites-available/nextcloud.conf
   ```

   Add the following content:
   ```apache
   Alias /nextcloud "/var/www/nextcloud/"

   <Directory /var/www/nextcloud/>
      Options +FollowSymlinks
      AllowOverride All

      <IfModule mod_dav.c>
         Dav off
      </IfModule>

      SetEnv HOME /var/www/nextcloud
      SetEnv HTTP_HOME /var/www/nextcloud
   </Directory>
   ```

   Enable the site:
   ```bash
   sudo a2ensite nextcloud.conf
   ```

   Enable necessary Apache modules:
   ```bash
   sudo a2enmod rewrite
   sudo a2enmod headers
   sudo a2enmod env
   sudo a2enmod dir
   sudo a2enmod mime
   ```

   Restart Apache:
   ```bash
   sudo systemctl restart apache2
   ```

7. **Finish Installation:**
   Open your web browser and navigate to `http://your_server_ip/nextcloud` to complete the Nextcloud installation.

Remember to replace `'your_password'` with a strong password for the database user. This is a simplified guide, and you may need to adjust some settings based on your specific requirements.
